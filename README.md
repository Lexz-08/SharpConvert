## SharpConvert
### Description
Can convert any Type to another.

### How To Use
```csharp
// Convert a number to a string without using Convert.ToInt32(string value);
string str = "0";
int num = str.ConvertTo<int>();

// For Console Application users...
Console.WriteLine(num);

// For Windows Forms Application users...
MessageBox.Show(num.ToString());
```
